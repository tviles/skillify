# /skillify:list

List all skillify-generated skills and commands across project and global locations.

## Instructions

1. Scan these directories for markdown files with `generated_by: skillify` in their YAML frontmatter:
   - `~/.claude/skills/` (global skills)
   - `~/.claude/commands/` (global commands)
   - `.claude/skills/` (project skills, if present)
   - `.claude/commands/` (project commands, if present)

   You can use the frontmatter parser:
   ```python
   import sys
   sys.path.insert(0, "${CLAUDE_PLUGIN_ROOT}")
   from core.frontmatter_parser import find_skillify_skills

   directories = [
       str(Path.home() / ".claude" / "skills"),
       str(Path.home() / ".claude" / "commands"),
       ".claude/skills",
       ".claude/commands",
   ]
   results = find_skillify_skills(directories)
   ```

2. For each file found, extract from frontmatter: name, confidence, generated_date.
   Infer type (skill vs command) from the file path. Skip any `.backup.md` files.

3. Present results in this format:

```
Skillify-generated skills and commands:

  Skills:
    name                     confidence  date        location
    react-perf-debugging     high        2026-04-03  global
    prisma-migration-flow    medium      2026-04-01  project

  Commands:
    run-integration-suite    high        2026-04-02  global

N items total (X skills, Y commands)
```

4. If no skillify-generated items are found, say:
   "No skillify-generated skills or commands found. Run `/skillify` to create one from this conversation."
